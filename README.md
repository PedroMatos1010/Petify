# Petify - Plataforma de Gestão Veterinária

**O Petify é uma solução "Full-Stack" para modernizar clínicas veterinárias.**
O projeto integra uma plataforma web de gestão clínica e uma aplicação móvel para os tutores dos animais.

---

## **Funcionalidades Principais**

### **Backoffice Web (React.js)**
* **Gestão Clínica:** Fichas de animais, histórico de vacinas e cirurgias.
* **Agenda Digital:** Marcação de consultas em tempo real.
* **Dashboard:** Visualização estatística da clínica.

### **Aplicação Móvel (React Native / Android)**
* **Para o Cliente:** Os donos podem ver o boletim de vacinas, marcar consultas e receber alertas.
* **Sincronização:** Tudo o que acontece na App reflete-se no Web Backoffice instantaneamente.

---

## **Stack Tecnológica**

| Componente | Tecnologia |

| **Mobile App** | React Native, Kotlin (Android) |
